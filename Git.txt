A new repo from an existing project:
	Go into the directory containing the project.
		1- git init.
		2- touch README.md
		3- git add .
		4- git add .gitignore		
		5- git commit -m 'first commit'
		6- git remote add origin git@github.com:msh8261/cicd_wine_quality
		7- git remote -v
		8- git status
		9- git push origin main
